How to Play

Select an aircraft for each square that meets the criteria for both the row and column value. Categories are independent, so that aircraft doesn't need to record both categories together.

For example, if the categories are Manufactured by Airbus and Operated by Emirates, the aircraft must be both currently in Emirates fleet and manufactured by Airbus.

There are no limits on the number of guesses per square.

You can change a submitted Aircraft by clicking the square again, even after the game is complete.

Rarities
